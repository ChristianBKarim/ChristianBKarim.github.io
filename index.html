Tile Painting System Integrated Into Sokoban Game Project

Developed a system for a Sokoban-Style Puzzle Game where players control a pawn which absorbs colors from special tiles, and must paint the desired color on all regular tiles in the level to win.

System Mechanics:

  • Actor placed in Level which changes game objective from reaching an end Tile to Painting all Regular Tiles.
  • Color-Orb Tiles which Paints the Pawn the same color as the Orb.
  • Painting Mechanic where Pawn will paint all Regular Tiles it walks accross the same color on itself.
  • Colored Gate Actors whcih prevents pawns from moving through unless painted the same color as the wall.
  • Introduction to the Level through UI, also informing the player of the current objective to paint all tiles to win.
  • Reminder of Desired Color by the level trough UI, Shown Constantly on Screen during gameplay.
